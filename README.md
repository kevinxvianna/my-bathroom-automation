# 🚧 [WIP] My Bathroom Automation 


## 🇧🇷 Português

### Sobre

Este repositório reúne códigos em Arduino voltados para automação residencial, com foco em controle de iluminação, dispositivos e integrações futuras.

O projeto ainda está em desenvolvimento (WIP), então a estrutura, os códigos e a documentação podem mudar com frequência.

### Projeto atual

O script `bathroom-mirror-light-strip-controller` é responsável por controlar uma fita LED RGBW da Stella (Stella All Light Ultra - Stl24879/rgbw), utilizada como iluminação de um espelho de banheiro.

A ideia é permitir controle de intensidade e cores de forma simples, servindo como base para evoluções e novos projetos dentro do repositório.

### Próximos passos

- Publicar um circuito no Tinkercad com a ligação dos componentes
- Evoluir para integração com automação (ex: apps, assistentes, etc.)

## 🇺🇸 English

### About

This repository contains Arduino-based projects focused on home automation, including lighting control, device interaction, and future integrations.

The project is currently a Work in Progress (WIP), so the structure, code, and documentation may change frequently.

### Current project

The `bathroom-mirror-light-strip-controller` script is responsible for controlling a Stella RGBW LED strip (Stella All Light Ultra - Stl24879/rgbw) used as lighting for a bathroom mirror.

The goal is to provide simple control over brightness and colors while serving as a foundation for future improvements and new projects.

### Next steps

- Add the electronic schematic
- Publish a Tinkercad project with the circuit connections
- Expand into automation integrations (e.g., apps, assistants, etc.)
